# Space Scarface — Frontend

## Стек
- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- NextAuth.js (авторизация)
- TanStack Query (запросы к API)

## Структура админки

### 1. Авторизация
- Страница логина (логин + пароль)
- Защищённые роуты через NextAuth

### 2. Генерация видео
Параметры:
- **Промпт** — текстовое описание видео
- **Длина** — секунды (15 / 30 / 60 / 90 / кастом)
- **Тематика** — категория контента (юмор, бизнес, мотивация и т.п.)
- **Соотношение сторон** — 16:9 (YouTube), 9:16 (TikTok/Reels), 1:1 (Instagram)
- **Стиль** — кинематограф, анимация, слайдшоу, новостной
- **Голос/озвучка** — язык, пол, тон
- **Фоновая музыка** — жанр, громкость
- **Субтитры** — вкл/выкл, язык
- **CTA** — текст призыва к действию
- **Логотип/Watermark** — загрузка файла

### 3. Просмотр результата
- Плеер с превью видео
- Кнопка "Опубликовать"
- Выбор платформ для публикации

### 4. Настройки платформ
Поддерживаемые платформы:
- Telegram (бот токен + chat_id)
- Instagram (логин/пароль или API токен)
- YouTube (OAuth)
- TikTok (OAuth)
- ВКонтакте (токен)

Для каждой платформы: название, credentials, статус подключения (активна/нет)
